# Audio Amplifier Design (OrCAD)

This repository contains the schematic design files for an audio amplifier project developed using **OrCAD Capture CIS**. 

### 📌 Project Contents & Note on Simulation Files
Due to GitHub's file size limitations, the raw PSpice simulation data files (which exceed 400 MB) could not be uploaded directly. Instead, you can find the complete project blueprints and results here:

* **Schematic Files:** Open the `.DSN` and `.opj` files directly in OrCAD to view the full circuit configuration.
* **Circuit Preview:** A quick visual layout is available in the `AMPLIFICATOR AUDIO.png` file.
* **Full Project Documentation:** For a detailed look at the simulation results, waveform graphics, component lists, and structural calculations, please refer to the **PDF or Word documentation** included in this repository *(Note: The documentation text is written in Romanian)*.
